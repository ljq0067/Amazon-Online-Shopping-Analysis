Keepa API

All the python scripts and raw data in this section can be found at data_gather/API_python/keepa folder on Github.

Keepa.com is a websites provides a nice API of information about products or sellers on Amazon for different regions or periods.

Firstly, please subscribe to the keepa API to get an access key. Then the historical price can be requested by providing the ASIN of product. The ASIN is a unique 10-characters identifier starts with 'B', which can be easily found in the Amazon url of the product. You can use the keepa API to request the ASIN for products on Amazon which sorted by sales ranking and then use the keepa API to request their information.

Assume the date March 19, 2020, when California issued statewide the first Stay-at-Home order, is the start of Covid-19 period, the question is the changes for online shopping during this time.

According to the ASIN list above, we can request the detail information about these products based on the selected period(before Covid-19 and after) and domain locale(.com) for amazon. The format of response and meaning of each value is showed below:

{
"productType": 0 for everything accessible, 1 for no marketplace/3rd party price data, 2 for no price data and sales rank accessible, 3 for no data accessible, 4 for no current data available due to invalid or deprecated asin, 5 for product of a parent ASIN,
"asin": The ASIN of the product,
"domainId": Product Amazon locale,
"title": Title of the product,
"trackingSince": The time started tracking this product,
"listedSince": The time the item was first listed on Amazon and 0 or -1 for not available,
"lastUpdate": The last time updated the information for this product,
"lastRatingUpdate": The last time updated the product rating and review count,
"lastPriceChange": The last time registered a price change (any price type),
"lastEbayUpdate": The last time updated the eBay prices for this product,
"imagesCSV": Amazon image names of the product in https://images-na.ssl-images-amazon.com/images/I/image name,
"rootCategory": Category node id of the product’s root category and 0 if no root category known. The id 9223372036854775807 denotes a blank category with the name “?”,
"categories": Array of Amazon category node ids this product is listed in,
"categoryTree": The category tree as an ordered array of objects containing category id and name,
"parentAsin": The ASIN of the parent product (if the product has variations),
"variationCSV": CSV list of up to 1800 variation ASINs of the product if available,,
"frequentlyBoughtTogether": One or two “Frequently Bought Together” ASINs,
"eanList": A list of EAN assigned to this product. The first index is the primary EAN,
"upcList": A list of UPC assigned to this product. The first index is the primary UPC,
"manufacturer": Name of the manufacturer,
"brand": An item’s brande,
"productGroup": The item’s productGroup,
"partNumber": The item’s partNumber,
"author": The item’s author,
"binding": The item’s binding. If the item is not a book it is usually the product category instead,
"numberOfItems": The number of items of this product. -1 if not available,
"numberOfPages": The number of pages of this product,
"publicationDate": The item’s publication date in one of the following three formats: YYYY or YYYYMM or YYYYMMDD,
"releaseDate": The item’s release date,
"languages": An item can have one or more languages and each language entry has a name and most also have a type,
"model": The item’s model,
"color": The item’s color,
"size": The item’s size,
"edition": The item’s edition,
"format": The item’s format,
"features": A list of up to 5 product features / bullet points,
"description": A description of the product,
"packageHeight": The package’s height in millimeter,
"packageLength": The package’s length in millimeter,
"packageWidth": The package’s width in millimeter,
"packageWeight": The package’s weight in gram,
"itemHeight": The item’s height in millimeter,
"itemLength": The item’s length in millimeter,
"itemWidth": The item’s width in millimeter,
"itemWeight": The item’s weight in gram,
"availabilityAmazon": 1 for no Amazon offer exists, 0 for Amazon offer is in stock and shippable, 1 for Amazon offer is currently not in stock but will be in the future (pre-order), 2 for Amazon offer availability is “unknown”, 3 for Amazon offer is currently not in stock but will be in the future (back-order), 4 for Amazon offer shipping is delayed,
"availabilityAmazonDelay": If availabilityAmazon has the value 4 this field will provide the delay interval in hours,
"ebayListingIds": Contains the lowest priced matching eBay listing Ids, the first one is the listing id of the lowest priced listing in new condition, the second in used condition,
"isAdultProduct": Whether the item is considered to be for adults only or not,
"launchpad": Whether the item is listed in the launchpad category or not,
"audienceRating": The age for which the media is appropriate,
"newPriceIsMAP": Whether the lowest new price (either price type AMAZON or NEW) is restricted by MAP (Minimum Advertised Price) or not,
"isEligibleForTradeIn": Whether the product is eligible for trade-in or not,
"isEligibleForSuperSaverShipping": Whether the product’s buy box is eligible for free shipping or not,
"fbaFees": Providing the FBA pick & pack fees of this product,
"variations": The dimension attributes for variations of this product,
"coupon": The first entry is the discount of a one time coupon, the second is a subscribed and save coupon,
"promotions": Contains the type of promotion, the discounted price and the discount percent,
"stats": Statistics Contains the current and historical price and out of stock percentage of the time,
"salesRankReference": Category node id of the main sales rank. -1 if not available. -2 if listed in launchpad,
"salesRanks": An object containing sub-category rank histories,
"rentalDetails": Contains the rental details description of the rental buy box offer,
"rentalSellerId": Contains the seller id of the rental buy box offer,
"rentalPrices": This object contains the rental prices like initialPrice, shortExtnPrice, longExtnPrice and fullPrice,
"csv": Contains the history data for the product
}

# Browsing

Setting the Amazon locale is .com, the sort order is the lowest rank to highest and the time interval in which the product changed is 3 months, choosing to search for Amazon price history, the browsing information can be found by Keepa API. The response contains a deals field with the following content: ordered array of all deal objects, root category ids and names of deal products, and how many deal products in the respective root category are found. Note that if the root category of a product could not be determined it will be listed in the category with the name “?” with the id 9223372036854775807.

Screenshot: Response JSON of browsing information from Keepa API

# Seller

For sellers in Amazon, the top sellers for Amazon locale(.com) can be searched and their id can be gotten from Keepa API like this.

Screenshot: Response JSON of top sellers from Keepa API

Through the seller id, the detail information can also be requested. The format of response is showed like below for each seller.

{
"domainId": ID of the Amazon locale,
"trackingSince": The time started tracking this seller,
"lastUpdate": The time of last update of this seller,
"sellerId": ID of sellers,
"sellerName": Name of sellers,
"isScammer": Whether our system identified that this seller attempts to scam users or not,
"shipsFromChina": Whether this seller ships from China or not,
"hasFBA": Fulfillment by Amazon(FBA) or not,
"totalStorefrontAsins": The number of storefront ASINs if available and the last update of that metric,
"csv": The merchant’s rating in percent at present and in history,
"asinList": Storefront ASINs, sorted by freshest first,
"asinListLastSeen": The last time (in Keepa Time minutes) we were able to verify each ASIN in the asinList field,
"sellerCategoryStatistics": Contains the category id, the number of products this merchant sells in this category, the 30-day average sales rank of these products, and how many of these products have an Amazon offer,
"sellerBrandStatistics": Contains the brand, the number of products this merchant sells with this brand, the 30-day average sales rank of these products, and how many of these products have an Amazon offer,
}
